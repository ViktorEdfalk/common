Funktioner

I Webcert kan du skapa och hantera intyg. Webcert stödjer utfärdande av vissa intygstyper för patienter med sekretessmarkering.

För Försäkringskassans intyg finns en elektronisk funktion för frågor och svar samt hantering av kompletteringsbegäran. Via funktionen kan läkaren och Försäkringskassan kommunicera om ett signerat intyg.

Invånarens intyg

I samband med att ett intyg signeras görs det tillgängligt för invånaren i Mina intyg, med undantag för Dödsbevis och Dödsorsaksintyg som inte visas i Mina intyg. I Mina intyg kan invånaren skicka intyget vidare elektroniskt till någon av de anslutna mottagarna eller skriva ut det. För sjukpenningintyget är det också möjligt att skapa en anpassad utskrift som kan ges till arbetsgivaren. Mina intyg nås via 1177 Vårdguidens e-tjänster eller via .

Om invånaren inte vill använda Mina intyg är det möjligt för läkaren att skicka intyget elektroniskt direkt från Webcert till huvudmottagaren av intyget. Observera att detta ska göras i samråd med patienten.

Läkaren kan också skriva ut det signerade intyget från Webcert och ge till patienten.

Webbläsare som stöds

För att kunna använda Webcerts funktioner behöver du Internet Explorer 11 eller efterföljande versioner. Det går att använda andra webbläsare men då finns viss risk för att problem uppstår. Du måste även ha JavaScript aktiverat i din webbläsare för att kunna använda Webcert.
