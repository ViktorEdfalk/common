Du har en sekretessmarkering. Var uppmärksam på vad det innebär vid användning av Webcert:

  • När du signerar ett intyg kommer ditt namn och information om den vårdgivare och vårdenhet intyget är utfärdat på, vara synligt.
  • Vid kommunikation med Försäkringskassan kring ett intyg, kommer ditt namn att vara synligt.

Detta kan göra att information om dig och var du arbetar kan spridas.
